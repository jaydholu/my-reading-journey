Flight Price Predictor

I also developed a Flight Price Predictor web app, hosted on Flight Price Predictor It uses machine learning models to predict the ticket prices of flights based on parameters like:

  • Departure & Arrival times
  • Duration of the flight
  • Airline and number of stops
  • Source and destination cities

This project combines practical data science with real-world applications, giving users quick estimates before booking a flight.

Technologies used: Python, Pandas, Scikit-learn, Flask, HTML/CSS, and Render for deployment.
